chara被设计为运行在Linux上的递归DNS服务器。

## 特点

1. 支持以TCP或UDP发起的DNS请求
2. 递归服务器
3. 可通过JSON形式配置解析结果
4. 提供组织DNS污染、主动刷新缓存功能
